Announcing Iter0

May 5, 2020 · Bharani

What is iter0

Iteration 0 is the first iteration of a software project. It is in this iteration we make important choices. These choices have a great impact on what gets built incrementally over time so we need to pay attention and make as fewer choices as possible and leave the rest for the last responsible phase of the project.

Iter0 is my attempt to distil the experience of evaluating software components during this iteration 0.

Why bother to document it here?

Writing this down clears my thinking and I hope someone would find this useful.
