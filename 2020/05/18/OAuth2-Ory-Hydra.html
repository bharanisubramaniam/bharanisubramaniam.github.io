What is OAuth2?

When you search for OAuth2 you come across definitions like

OAuth2 is the industry-standard protocol for authorization

We usually relate authorization to a framework that enables us to control the allowed list of actions that a user can perform.We implicitly assume that the user is first authenticated into the system before these authorization rules can be enforced.

OAuth2 is not that authorization framework we just described.So the first definition needs some refinement - here is my first attempt

OAuth2 is the industry-standard protocol to authorize access to an user's account

If you are a beginner this definition may raise more questions so let's quickly cover the needs that drove the industry towards OAuth2

What led to OAuth2?

Users have to explicitly setup authentication mechanisms in each and every system

Two main problems with this setup is that

  1. Users have to set up their identity multiple times in different systems
  2. It is not easy to let System B access resources of System A for the same user.
